# Carpeta de imágenes

Coloca aquí las dos fotos para el juego:

- `debora.jpg` → Foto de Débora (será la cara del personaje que salta)
- `lissy.jpg` → Foto de Lissy (aparece al final del nivel Difícil)

Recomendaciones:

- Usa fotos donde la cara esté centrada (se recortan en círculo).
- Tamaño recomendado: entre 400x400 y 800x800 píxeles.
- Formato: `.jpg` o `.png` (cambia el nombre del archivo si usas `.png`).

Si las fotos no están, el juego usa una carita kawaii dibujada como respaldo, así que la web nunca se rompe.
